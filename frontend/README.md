# Travel Agent Frontend

React, TypeScript, Vite 기반 여행 계획 UI입니다.

```bash
npm install
npm run dev
npm run test
npm run build
```

환경 변수:

```bash
VITE_API_BASE_URL=http://127.0.0.1:8000
```

기본 개발 URL은 `http://localhost:5173`입니다. 해당 포트가 이미 사용 중이면 Vite가 다른 포트로 이동할 수 있으므로 백엔드 `CORS_ORIGINS`에도 해당 origin을 추가해야 합니다.
